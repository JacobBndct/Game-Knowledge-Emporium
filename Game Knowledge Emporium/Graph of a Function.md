*The **graph** of $f : X \rightarrow Y$ is defined as* $$G(f) = \{(x,y) \in X \times Y : y = f(x)\}$$
*For example the graph of the function $f : [2] \rightarrow \text{ stars given by } 0 \mapsto \star, 1 \mapsto \ast$ is $G(f) = \{(0, \star), (1, \ast)\}$.*
