*For a function $f: X \rightarrow Y$ and a subset $T \subseteq Y$ define the **inverse image** $f^{-1}(T)$ to be the set of all elements of $X$ that are mapped to $T$. in other words $f^{-1}(T) = \{ x \in X : f(x) \in T\}$.*
